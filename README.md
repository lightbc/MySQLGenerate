# MySQLGenerate
MySQL代码生成工具。
# 用途
用于生成MySQL数据库的实体类文件。
# 使用/提供
输入连接的数据库地址、用户名、密码，连接数据库，通过操作按钮功能提示进行操作。提供了表名快速查询选择，用于快速生成指定表功能。提供将本次连接信息进行保存操作，用于下次快捷连接使用，避免重复输入的麻烦。
